## 🎯 Why Mallet?

Hammer is heavy, outdated, and crashes at every clumsy move.
**Mallet** is built from scratch with modern C++ and Qt – lighter, faster, and more reliable.


| Feature | Hammer | Mallet |
|---------|--------|--------|
| 🚀 Launch time | 45 seconds | ~20 seconds |
| 💥 Random crashes | Yes | Very rare |
| 🧠 Smart grid snapping | Basic | Advanced |
| 🎨 Modern UI | WinForms‑style | Qt6 (Dark/Light themes) |
| 📦 Built‑in compiler | VVIS / VRAD | Greatly improved VVIS/VRAD |
| 🧵 Multi‑threading | No | Yes (Qt concurrent) |

---

## ✨ Features

- **Fast level editing** – no more waiting for brushes to load.
- **Real‑time lighting preview** – what you see is what you compile.
- **Intelligent texture alignment** – auto‑detects seams & fixes them.
- **Built‑in model browser** – drag & drop assets directly into the scene.
- **Live collaboration** – edit maps with your team in real time.
- **Cross‑platform** – Windows, Linux, macOS (single codebase).
